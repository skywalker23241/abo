## Overview

**Abo Blog** is my personal blog repository, built with **Astro** and focused on clean writing, technical notes, SEO practice, and lightweight web design.

The site is designed to feel simple, quiet, and fast — like a black-and-white signal log from a small station somewhere in deep space.

```txt
PROJECT       Abo Blog
SITE          https://jackcooper.qzz.io/
AUTHOR        Junbo Le / Abo
STACK         Astro / MDX / TypeScript / CSS
FOCUS         Blogging / SEO / Web Development / Personal Notes
DEPLOYMENT    Netlify
```

---

## Features

- Minimal personal blog layout
- Astro-powered static site architecture
- MDX support for flexible writing
- SEO-friendly structure with sitemap and robots support
- Reading time, table of contents, and copy-code support
- KaTeX support for math rendering
- Image processing and cleanup pipeline
- Link cards and embedded media support
- Fast page transitions and optimized build output

## Getting Started

### 1. Install dependencies

```bash
pnpm install
```

### 2. Start the development server

```bash
pnpm dev
```

### 3. Build for production

```bash
pnpm build
```

### 4. Preview the production build

```bash
pnpm preview
```

---

## Scripts

| Command | Description |
| --- | --- |
| `pnpm dev` | Start the local Astro dev server |
| `pnpm build` | Build the site for production |
| `pnpm preview` | Preview the production build locally |
| `pnpm lint` | Run ESLint checks |
| `pnpm lint:fix` | Fix lint issues automatically |
| `pnpm format` | Format files with Prettier |
| `pnpm format:check` | Check formatting |
| `pnpm new` | Create a new blog post |
| `pnpm update-theme` | Update theme-related files |
